## Commands

### grok chat
Start an interactive chat session.

### grok code
Generate or modify code.

### grok acp
Run in ACP mode for editor integration.

### grok skills
Manage skills.

- grok skills list: List available skills.

- grok skills show <name>: Show details of a skill.

- grok skills new <name>: Create a new skill template.

- grok skills validate <name>: Validate a skill for security.

- grok skills enable <name>: Enable a skill.

- grok skills disable <name>: Disable a skill.

- grok skills generate-docs: Generate this documentation.

### grok config
Manage configuration.

### grok settings
View and modify settings.

### grok setup
Setup the CLI.

### grok health
Check system health.

### grok history
View command history.

### grok audit
Audit logs.

## Configurations

Configuration is managed via `config.toml` in the `.grok` directory.

See `config.example.toml` for available options.
